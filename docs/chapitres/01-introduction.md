# Introduction

Voici quelques préconisations et bonnes pratiques, pleines de bon sens,
à suivre lorsque vous déployez des applications 4D, agrémentées
d'informations techniques.

Attention : certaines préconisations doivent se faire dans des contextes
précis car une mauvaise compréhension pourrait générer des problèmes.
Merci de vous rapprocher de votre interlocuteur technique 4D pour plus
d\'informations.
