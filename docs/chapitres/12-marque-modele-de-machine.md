# Marque / Modèle de machine

Nous n\'avons pas de préconisations particulières, il faut cependant
regarder en détails le matériel qui compose la machine avant de
l\'acheter afin de s\'assurer que les composants soient compatibles
entre eux et que ses caractéristiques correspondent à vos attentes et
aux préconisations ci-dessus.
