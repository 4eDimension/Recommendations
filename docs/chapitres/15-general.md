# Général

Si votre application 4D a été créée avec une très ancienne version de
4D, il est possible qu'une zone « Priorités CPU » soit visible dans
l'onglet « Général » des propriétés de votre base de données.

Ce paramétrage est désormais obsolète. Lorsque cette zone est affichée,
vous devez absolument cliquer sur le bouton « Réinitialiser aux réglages
d'usine » afin de réinitialiser les paramètres et de les supprimer de
la boîte de dialogue.
