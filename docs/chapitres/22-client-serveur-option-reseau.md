## Client-Serveur / Option réseau

- Cocher « Publier la base au démarrage »

- Renseigner le numéro de port de publication des données

- Décocher l'option « Inscrire les clients au démarrage » si vous
  n'utilisez pas la commande 4D « Exécuter sur client »

- Cocher l'option « Crypter les communications Client-Serveur »
