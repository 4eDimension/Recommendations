## Web / Options (I)

- Cocher « Utiliser le cache Web de 4D »

- Définir la taille du cache des fichiers statiques (524 288 Ko par
  défaut)

- Cocher l'option « Sessions extensibles » pour gérer plusieurs requêtes

- Sélectionner un utilisateur Web générique

Si vous utilisez une ancienne version de 4D :

- Fixer à 8 heures le délai de conservation des process inactifs et
  cocher la case « gestion automatique des sessions » pour que les
  utilisateurs Web puissent réutiliser le même contexte durant la
  journée (si vous ne les gérez pas par programmation)
