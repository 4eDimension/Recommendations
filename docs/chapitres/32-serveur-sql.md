# Serveur SQL

Alors que l'accès aux données pour un client 4D utilise par défaut un
protocole propriétaire, l'accès SQL (natif ou via ODBC) est aussi bien
pris en charge. En outre, il existe des drivers PDO Open Source
disponibles (PHP Data Objects). L'accès SQL au niveau base de données
peut être contrôlé par un système de mot de passe, des schémas SQL ou à
l'aide de vues SQL.
