# Backup / Backup & restore

- Check the \"Keep the last X backup files\" option

- Define the number of backup sets to keep

- Select \"After backup\" to erase the oldest backup

- Check \"Retry in 561 seconds\" if the backup fails

- Uncheck the \"Cancel operation after X attempts\" option

- Check the option \"Restore the last backup if the database is damaged

- Check the option \"Integrate the last history if the database is
  incomplete\".

- Select the \"Strict mode\" for the integration option
