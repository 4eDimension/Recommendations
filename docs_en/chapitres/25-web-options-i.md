# Web / Options (I)

- Check \"Use 4D Web cache\"

- Define the size of the page cache (524 288 Kb by default)

- Check the \"Expandable Sessions\" option to manage multiple requests

- Select a generic Web user

If you are using an older version of 4D:

- Set the retention time for inactive processes to 8 hours and check the
  \"Automatic session management\" box so that Web users can reuse the
  same context during the day (if you do not manage them
  programmatically)
