# Compatibility

Concerns only converted databases (from a previous version of 4D).

- Check \"Execute SEARCH BY FORMULA\" on the server

- Check \"Execute SORT BY FORMULA on the server

- Check \"SEARCH BY FORMULA uses SQL joins\"

- Uncheck \"Automatic assignment of variables

- Uncheck \"Use old network layer

- Check \"Allow nested transactions

- Check \"Use Unicode mode

- Check \"Define period and comma as location characters in numeric
  formats

- Check \"Use object notation to access object properties
