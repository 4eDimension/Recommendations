# Software update mechanism

Modern software is a complex combination of software, database server,
middleware, application server, web server and more. It is easy to
forget to keep all the pieces up to date, like an OpenSSL DLL for
example. 4D reduces this problem in many ways, not only by helping the
administrator in his daily life, but also by reducing the risk by
design.

With everything integrated into one solution, there is only one folder
to replace. Everything is installed in one folder, it could even be
replaced by a drag and drop process. Keeping it simple avoids the
\"I'll do it later\" syndrome. With a single replace, all parts of the
business application are updated in one step, nothing can be missed.

The server can be updated automatically. The update process is
controlled and forced by 4D itself, so it can be driven by the solution
developer.
